# Scanning Ministry  
## Runtime / Ministries

---

## 1. Purpose

The Scanning Ministry gathers information from the environment, devices, and APIs.  
It provides situational awareness to the system.

---

## 2. Responsibilities

- device scanning  
- API scanning  
- sensor input  
- environmental awareness  
- data validation  

---

## 3. Boundaries

The Scanning Ministry does not:

- interpret meaning  
- execute actions  
- manage user experience  

It collects information; it does not decide what it means.

---

## 4. Constitutional Statement

**Scanning must be accurate, safe, and transparent.**
